### Quick start

- Dev: `npm run dev`
- Typecheck: `npm run typecheck`
- Build: `npm run build`

### Conventions

- All import paths use `@/` which maps to `src/`.
- Components use React 19 and Tailwind CSS 4.
- Data fetching uses TanStack Query v5.
